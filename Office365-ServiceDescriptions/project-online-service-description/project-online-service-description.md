# <a name="microsoft-project-service-description"></a>Описание службы Microsoft Project

## <a name="overview"></a>Обзор

Microsoft Project предлагает следующие приложения для удовлетворения потребностей организации в управлении проектами и работой:

- Project в Интернете
- Project Online 
- Клиент для настольных компьютеров Project Online

В этой статье вы сможете понять, какие приложения предоставляются в каждой подписке на Project и какие возможности предлагает каждое приложение.

### <a name="project-for-the-web"></a>Project в Интернете

Project for the web — это последнее предложение Корпорации Майкрософт для облачной работы и управления проектами. Project for the web предоставляет простые и мощные возможности управления работой для удовлетворения большинства потребностей и ролей. Руководители проектов и члены группы могут использовать Project для веб-планирования и управления работой любого размера.

Проект для Интернета построен на платформе Microsoft Power Platform. Платформа Power состоит из PowerApps, Power Automate, Power BI и Microsoft Dataverse. Project для веб-данных хранится в Microsoft Dataverse. 

Проект для Интернета доступен по трем подпискам: Project Plan 1, Project Plan 3 и Project Plan 5. [Функции, перечисленные в Project для веб-функций,](#project-for-the-web-features) взаимодействуют только с данными в Project для веб-магазина данных.

### <a name="project-online"></a>Project Online
 
Project Online — это гибкое онлайн-решение для управления портфелями проектов (PPM) и повседневной работы. Project Online предоставляет мощные возможности управления проектами для планирования, приоритизации и управления проектами и инвестициями в портфель проектов практически из любой точки на &mdash; любом устройстве. Project Online может использоваться администраторами, руководителями и обозревателями портфелей, руководителями проектов и ресурсов, руководителями и участниками групп.

Project Online построен на платформе SharePoint и хранит данные в хранилище данных SharePoint.

Project Online доступен по двум подпискам: Project Plan 3 и Project Plan 5. Функции, [перечисленные в функции Project Online,](#project-online-features) взаимодействуют только с данными в хранилище данных Project Online SharePoint.

Функциональные возможности членов группы project Online доступны в подписке Project Online Essentials или Project Plan 1. Project Online Essentials —  это подписка на надстройки членов группы для клиентов, которые имеют подписки Project Plan 3 или Project Plan 5. Подписки Project Online Essentials и Project Plan 1 дают участникам группы веб-интерфейс для выполнения обновлений задач и времени для проектов, управляемых в Project Online.

Участники группы с подписками на Project Online Essentials или Project Plan 1 могут выполнять следующие функции Project Online:

- Использование веб-интерфейса
- Обновление задач, проблем и рисков
- Отправка расписаний
- Совместное участие в работе с документами и совместное взаимодействие с пользователями Microsoft Teams или Skype для бизнеса

Project Online Essentials не включает Project for the Web. Project Plan 1 включает в себя как Project for the web, так и Project Online Essentials.

### <a name="project-online-desktop-client"></a>Клиент для настольных компьютеров Project Online

Настольный клиент Microsoft Project Online — это программа управления проектами, которая включает автоматическое планирование, управление ресурсами проекта и встроенную отчетность. Клиент для настольных компьютеров Project Online может использоваться в качестве автономных приложений или подключаться к Project Online. Клиент настольного компьютера Project Online — это версия подписки для настольного клиента Project Professional, которая доступна в подписках Project Plan 3 и Project Plan 5.

### <a name="using-project-for-the-web-and-project-online-together"></a>Совместное использование Project для веб-сайта и Project Online

Несмотря на то, что Project для веб-сайта и Project Online являются отдельными приложениями, их можно использовать рядом. Все проекты можно увидеть в обоих приложениях в Project Home. Вы можете использовать возможность Project Roadmap для создания дорожных карт, которые включают проекты из обоих приложений. Вы можете использовать Power BI и Project для пакета контента веб-power BI для получения информации о проектах и ресурсах в двух приложениях. <sup>3</sup>

На этой схеме показано, как приложения могут работать вместе друг с другом.

![Схема, показывающая проект в Интернете и Project Online вместе](../media/project-online-applications.png)

В будущем Корпорация Майкрософт сосредоточится на Project для веб-инноваций. Тем не менее, вы можете продолжать использовать Project Online с уверенностью, так как он будет продолжать получать ключевые улучшения производительности и безопасности.

## <a name="service-considerations"></a>Соображения службы

> [!NOTE]
> Проект для Интернета в настоящее время недоступен для GCC, GCC High и DoD. Мы работаем над доставкой Project для веб-сайта для вас, но в настоящее время не удается поделиться с вами твердыми таймфреймами.

### <a name="licensing-considerations"></a>Принципы лицензирования

- Любое взаимодействие на сайте Project Online требует по крайней мере подписки project Plan 3 или Project Plan 5 в клиенте.
- Когда истекает срок действия последней подписки на Project Plan 1, Project Plan 3 или Project Plan 5, ваш проект для веб-экземпляров не будет автоматически удален до тех пор, пока у вас не будет активных подписок, которые зависят от microsoft Dataverse.
- По истечении срока действия последней подписки на Project Plan 3 или Project Plan 5 экземпляры Project Online будут удалены через 120 дней.
- Для пробных подписок Project Online пробные экземпляры будут удалены через 30 дней после окончания пробного периода.
- Для Project для веб-пробных подписок пробные экземпляры не будут удалены до тех пор, пока у вас не будет активных подписок, которые зависят от Microsoft Dataverse.

### <a name="use-of-sharepoint-online"></a>Использование SharePoint Online

Project Online требует использования SharePoint Online, который является частью Project Online. Права на функции SharePoint Online, предоставляемые подписками Project Plan 3 или Project Plan 5, ограничены хранением и доступом к данным для поддержки Project Online.

### <a name="project-roadmap-and-power-automate"></a>Project Roadmap и Power Automate

Проектная дорожная карта требует использования Power Automate, которая предусмотрена в рамках подписки на Project. Права на функциональность Power Automate ограничены возможностями Power Automate, требуемой проектной дорожной картой. Функция Power Automate, требуемая проектной дорожной картой, отображается как интеграция данных для project с планом обслуживания **Flow** в разделе **Приложения** центра администрирования Microsoft 365.

### <a name="project-for-the-web-and-microsoft-dataverse"></a>Project for the web and Microsoft Dataverse

Project for the web требует использования microsoft Dataverse для хранения данных. База данных Microsoft Dataverse создана в рамках подписки на Project. Функциональность Microsoft Dataverse, требуемая  Project, отображается в качестве общей службы данных для плана службы project в разделе **Приложения** центра администрирования Microsoft 365.
