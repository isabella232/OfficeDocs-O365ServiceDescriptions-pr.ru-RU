# <a name="microsoft-project-service-description"></a>Описание службы Microsoft Project

## <a name="overview"></a>Обзор

Microsoft Project предлагает следующие приложения, которые помогут удовлетворить потребности вашей организации в управлении проектами и работой:

- Project в Интернете
- Project Online 
- Project Online настольный клиент

Эта статья поможет вам понять, какие приложения предоставляются в каждой Project и какие возможности предлагает каждое приложение.

### <a name="project-for-the-web"></a>Project в Интернете

Project для Интернета — это последнее предложение корпорации Майкрософт по облачной работе и управлению проектами. Project для Интернета обеспечивает простые, мощные возможности управления работой для удовлетворения большинства потребностей и ролей. Project менеджеры и члены команды могут использовать Project для Интернета для планирования и управления работой любого размера.

Project для Интернета построен на платформе Microsoft Power Platform. Платформа Power состоит из PowerApps, Power Automate, Power BI и Microsoft Dataverse. Project данных хранится в Microsoft Dataverse. 

Project для Интернета доступна через три подписки: Project, план 1, Project, план 3 и Project, план 5. Функции, перечисленные [Project для веб-функций,](#project-for-the-web-features) взаимодействуют только с данными в Project для хранения данных в Интернете.

### <a name="project-online"></a>Project Online
 
Project Online является гибким онлайн-решением для Project управления (система УПП) и повседневной работы. Project Online предоставляет мощные возможности управления проектами для планирования, определения приоритетов и управления проектами и портфельными инвестициями &mdash; практически из любой точки мира практически на любом устройстве. Project Online может использоваться администраторами, руководителями и обозревателями портфелей, руководителями проектов и ресурсов, руководителями и участниками групп.

Project Online построен на платформе SharePoint, и он хранит данные в SharePoint данных.

Project Online доступна через две подписки: Project, план 3 и Project, план 5. Функции, перечисленные [Project Online функции,](#project-online-features) взаимодействуют только с данными в Project Online SharePoint данных.

Функциональность участника команды для Project Online доступна через подписку Project Online базовый или Project, план 1 команды. Project Online базовый является подпиской на *добавление в команду* для клиентов, которые имеют Project, план 3 или Project, план 5 подписки. Project Online базовый и Project, план 1 предоставляет членам команды веб-интерфейс для выполнения задач и обновления времени для проектов, управляемых в Project Online.

Члены команды с Project Online базовый или Project, план 1 подписками могут выполнять следующие Project Online функции:

- Использование веб-интерфейса
- Обновление задач, проблем и рисков
- Отправка расписаний
- Делитесь документами и сотрудничайте с Microsoft Teams или Skype для бизнеса пользователями

Project Online базовый не включает Project для Интернета. Project, план 1 включает в себя Project для интернета и Project Online базовый.

### <a name="project-online-desktop-client"></a>Project Online настольный клиент

Microsoft Project Online настольный клиент — это программа управления проектами, которая включает автоматизированное планирование, управление ресурсами проекта и встроенную отчетность. Project Online настольный клиент может быть использован в качестве автономного приложения или он может подключиться к Project Online. Project Online настольный клиент является подписной версией настольного Project профессиональный и доступен в Project, план 3 и Project, план 5 подписках.

### <a name="using-project-for-the-web-and-project-online-together"></a>Использование Project для Интернета и Project Online вместе

Хотя Project для Интернета и Project Online отдельные приложения, вы можете использовать их бок о бок. Вы можете увидеть все ваши проекты из обоих приложений в Project Home. Вы можете использовать возможности Project дорожной карты для создания дорожных карт, которые включают проекты из обоих приложений. И вы можете использовать Power BI и Project для веб-Power BI контента, чтобы получить представление о проектах и ресурсах в двух приложениях. <sup>3 М.В.</sup>

На этой диаграмме показано, как приложения могут работать вместе бок о бок.

![Диаграмма, показывающая Project в Интернете и Project Online вместе](../media/project-online-applications.png)

В будущем корпорация Майкрософт сосредоточится на Project для веб-инноваций. Тем не менее, вы можете продолжать Project Online с уверенностью, поскольку он будет продолжать получать ключевые улучшения производительности и безопасности.

## <a name="service-considerations"></a>Соображения обслуживания

> [!NOTE]
> Project для Интернета в настоящее время недоступна для GCC, GCC High и DOD. Мы работаем над доставкой Project для Интернета для вас, но не в состоянии поделиться твердым сроки в настоящее время.

### <a name="licensing-considerations"></a>Принципы лицензирования

- Любое взаимодействие на Project Online требует, по крайней Project, план 3 или Project, план 5 подписки в пределах арендатора.
- Когда срок действия вашей последней Project, план 1, Project, план 3 или Project, план 5 истекает, ваш Project для веб-экземпляров не будет автоматически удален до тех пор, пока у вас не будет активных подписок, зависящих от microsoft Dataverse.
- Когда срок действия Project, план 3 или Project, план 5 истекает, Project Online экземпляры будут удалены через 120 дней.
- Для Project Online пробных версий пробные экземпляры будут удалены через 30 дней после окончания пробного периода.
- Для Project подписки на веб-пробные версии, ваши пробные экземпляры не будут удалены до тех пор, пока у вас не будет активных подписок, зависящих от Microsoft Dataverse.

### <a name="use-of-sharepoint-online"></a>Использование SharePoint Online

Project Online требует использования SharePoint Online, которая предусмотрена в рамках Project Online. Права на функциональность SharePoint Online, предоставляемые Project, план 3 или Project, план 5, ограничиваются хранением и доступом к данным для поддержки Project Online.

### <a name="project-roadmap-and-power-automate"></a>Project Дорожная карта и Power Automate

Project Дорожная карта требует использования Power Automate, которая предусмотрена как часть вашей Project подписки. Права на Power Automate ограничены теми возможностями, Power Automate, которые требуются Project карте. Power Automate, требуемая Project дорожной картой, отображается **как интеграция данных Project Flow с планом** обслуживания **приложений** в разделе приложений Microsoft 365 центра.

### <a name="project-for-the-web-and-microsoft-dataverse"></a>Project для Интернета и Microsoft Dataverse

Функция Microsoft Dataverse, требуемая Project, отображается как **Общая служба данных для Project** обслуживания в разделе **Приложения** Microsoft 365 центра.

Для Project клиентов с пятью (5) или более Project для веб-лицензий, Project для веб-клиентов могут развертываться в средах производства power Platform и Sandbox.
