# <a name="exchange-online-service-description"></a>Описание службы Exchange Online

Microsoft Exchange Online — это размещенное решение для обмена сообщениями, которое доставляет электронную почту, календарь, контакты и задачи с компьютеров, интернета и мобильных устройств. Он полностью интегрируется с Azure Active Directory, позволяя администраторам использовать групповые политики, а также другие средства администрирования для управления функциями Exchange Online в своей среде.

Организации, которые подписываются на Exchange Online, сохраняют контроль над службами обмена сообщениями, которые они предлагают пользователям. При использовании размещенных планов Exchange Online, описанных в этом документе, электронная почта размещается на серверах, которые одновременно поддерживают несколько клиентов. Эти серверы находятся в центрах обработки данных Майкрософт и доступны пользователям на различных устройствах из корпоративной сети или через Интернет.

## <a name="available-plans-for-exchange-online"></a>Доступные планы для Exchange Online

Microsoft 365 доступны в различных планах в соответствии с потребностями вашей организации. Подробные сведения о планах, которые позволяют пользователям использовать Exchange Online, см. в полной [таблице сравнения подписок](https://go.microsoft.com/fwlink/?linkid=2139145).

> [!NOTE]
> Каждый пользователь, который Exchange Online службу, должен быть назначен плану подписки, а у каждой подписки пользователя есть свой почтовый ящик. Папки и сообщения в этих почтовых ящиках находятся в центре обработки данных Майкрософт. Для конференц-залов и общих почтовых ящиков подписки пользователей не требуются. Эти специальные типы почтовых ящиков не имеют учетных данных для входа. Вместо этого лицензированные пользователи с соответствующими разрешениями управляют ими и доступом к ним через делегирование. Microsoft 365 F1 не включает права на Exchange почтового ящика. Чтобы обеспечить полный Teams, Microsoft 365 F1 лицензии могут включать Exchange Online плана обслуживания K1. Хотя план Exchange Online K1 подготавливает почтовый ящик для пользователя, Microsoft 365 F1 пользователи не имеют права использовать этот почтовый ящик. Мы рекомендуем отключить Outlook в Интернете этих действий и попросить пользователей не [](/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app) получать доступ к почтовому ящику Exchange другими способами.

## <a name="additional-services"></a>Дополнительные службы

### <a name="scheduler-for-microsoft-365"></a>Планировщик для Microsoft 365

Планировщик Microsoft 365 — это надстройка для Exchange Online, которая позволяет пользователям делегировать свои потребности в планировании цифровому личному помощнику. Помощник может планировать и перепланировать встречи и собрания с пользователями внутри и за пределами организации. Почтовый ящик помощника настраивается и управляется клиентом. Чтобы включить планировщик, администраторам необходимо настроить почтовый ящик помощника с помощью командлета PowerShell и приобрести лицензии для организаторов собраний. Дополнительные сведения о планировщике и его работе см. в статье ["Добро пожаловать в планировщик Microsoft 365](/microsoft-365/scheduler/scheduler-overview#how-does-scheduler-for-microsoft-365-work)". Сведения о ценах и лицензировании планировщика см. в [разделе Microsoft 365 лицензирования](https://www.microsoft.com/microsoft-365/meeting-scheduler-pricing).
