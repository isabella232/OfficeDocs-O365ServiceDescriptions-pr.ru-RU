# <a name="excel-for-the-web"></a>Excel в Интернете

Excel веб-страницы (ранее Excel Веб-приложение) расширяет Microsoft Excel веб-браузер, где вы можете работать с книгами непосредственно на веб-сайте, где хранится книга. Все клиенты могут просматривать и легко Office файлы с Office веб-страниц.
  
> [!NOTE]
> Книги с объемом более 50 мегабайт (МБ) не могут просматриваться в Excel веб-страницы из Microsoft Office SharePoint Online. Для просмотра объемных файлов необходимо установить на компьютер классическое приложение Excel. 

> [!NOTE]
> Чтобы помочь вам сравнить предложения, расширенные функции, перечисленные в этой статье, включают Excel возможности настольных приложений, доступные только в Office-пакете (например, Office профессиональный плюс, Office стандартный или Приложения Microsoft 365 для предприятий). Список всех Office веб-функций, таких как Word для веб PowerPoint и веб Office для [веб-планов.](office-online-service-description.md#feature-availability-across-office-for-the-web-plans)

Чтобы узнать, какой тип файла поддерживается Excel веб-сайте, см. в Office поддерживаемых типов [файлов для Веб.](office-online-service-description.md#supported-file-types-for-office-for-the-web) Дополнительные информацию о различиях между Excel веб-сайтом и настольным приложением Excel см. в книге Различия между использованием книги в браузере и [Excel](https://support.office.com/article/f0dc28ed-b85d-4e1d-be6d-5878005db3b6).

## <a name="power-view-viewing"></a>Power View viewing

Вы можете просматривать и взаимодействовать с Power View в книге, но вы не можете создать Power View с Excel веб-страницы. Создание Power View доступно только с помощью классического приложения. Power View позволяет создавать данные, предоставлять к ним доступ и анализировать их с помощью интерактивных представлений, готовых к презентации. Просматривайте информацию интерактивных карт, диаграмм и таблиц, связанных друг с другом, на одном листе. Узнайте больше о [Power View](https://go.microsoft.com/fwlink/p/?LinkId=271674).

Дополнительные сведения о совместном редактировании в реальном времени см. в видео [Одновременная работа нескольких пользователей над книгами Excel с помощью функции совместного редактирования](https://go.microsoft.com/fwlink/?linkid=850793).
