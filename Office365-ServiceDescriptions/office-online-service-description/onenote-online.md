# <a name="onenote-for-the-web"></a>OneNote в Интернете

Используйте OneNote для веб-сайта (ранее OneNote Web App) для записи в интернете в записной книжке OneNote, которую можно добавить из любой точки и легко делиться с другими. Все клиенты могут просматривать и легко изменять файлы Office с помощью Office в Интернете. 
  
## <a name="advanced-collaboration-new-content-shows-as-unread-presence"></a>Расширенные возможности совместной работы. Отображение нового содержимого как непрочитанного; сведения о присутствии

Недоступна в OneNote для интернета. В классическом приложении OneNote можно легко определить и проверить подлинность других авторов с помощью интегрированных профилей. Поиск в записной книжке изменений, редакций по именам авторов и просмотр всех недавних изменений при возвращении в общую записную книжку.
  
## <a name="apply-tags"></a>Применение тегов

С помощью OneNote для Интернета можно применить различные теги к заметкам для простой организации и последующей работы. Например, можно пометить заметки как вопросы, задачи или контактные данные. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="office-add-ins"></a>Надстройки Office

OneNote для интернета поддерживает только надстройки контента для Office. Контентные надстройки интегрируют веб-функции как содержимое, которое может быть показано в документе. Дополнительные сведения о [типах надстроек для Office](/previous-versions/office/office-2013-resource-kit/jj219429(v=office.15)).
  
## <a name="audio-notes-recording"></a>Запись звуковых заметок

Недоступна в OneNote для интернета. В классическом приложении OneNote можно записывать аудио- и видеозаметки, напрямую связанные с любыми текстовыми заметками, которые вы делаете во время записи. OneNote для Интернета не играет аудио- и видеоконтент, но носители сохраняются в записной книжке, и для воспроизведения их на компьютер можно скачать аудио- и видеофайлы. 
  
## <a name="bullets-and-numbering"></a>Маркеры и нумерация

С Помощью OneNote для Интернета можно применить выбор из трех стилей пули или пяти стилей номеров. Используйте кнопки Increase Indent и Decrease Indent для изменения уровня списка для существующих списков с пулями и номерами в записной книжке, а также для списков, созданных в OneNote для Интернета. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="clipboard"></a>Буфер обмена

С помощью OneNote для Интернета можно вырезать, скопировать и вклеить содержимое в блокнот. Можно скопировать и вклеить текст между OneNote для веб-сайта и настольным приложением OneNote. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="clipping-experiences"></a>Вырезка

Недоступна в OneNote для интернета. С помощью настольного приложения OneNote вы можете обрезать все, что видите на экране, отправить веб-страницу или весь документ в раздел записной книжки или прислать быстрые заметки, автоматически сохраненные и поданных в записной книжке.
  
## <a name="co-authoring"></a>Совместное редактирование

С Помощью OneNote для Интернета можно одновременно редактировать записные книжки с людьми в других местах, которые используют OneNote для веб-приложения или настольного приложения OneNote. Узнайте больше о [совместной работе в записной книжке OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272945).
  
## <a name="create-and-manage-pages-and-sections"></a>Создание страниц и разделов и управление ими

С помощью OneNote для Интернета можно легко добавить новые или существующие записные книжки, создав новые страницы в разделах или новые разделы в блокнотах. Страницы и разделы можно изменять, форматировать и перемещать или копировать между записными книжками. Узнайте больше об [использовании OneNote для веб-записной книжки.](https://go.microsoft.com/fwlink/p/?LinkId=272945)
  
## <a name="download-copy-of-embedded-office-files"></a>Загрузка копии встроенных файлов Office

С помощью OneNote для Интернета можно вставить файлы Office в виде вложений или распечаток, чтобы хранить их в записной книжке. Классическое приложение OneNote 2013 добавляет возможность внедрять электронные таблицы Excel и рисунки Visio в заметки. Узнайте больше о [внедрении файлов Microsoft Office в OneNote](https://go.microsoft.com/fwlink/p/?LinkId=271843).
  
## <a name="dropbox"></a>Dropbox

Dropbox — служба размещения файлов с облачным хранилищем, синхронизацией файлов, личным облаком и клиентским программным обеспечением.
  
## <a name="edit-embedded-files"></a>Редактирование внедренных файлов

С помощью OneNote для Интернета можно скачать встроенные, поданных на компьютер, чтобы открыть их. В классическом приложении OneNote можно вкладывать практически любой компьютерный файл в любую часть заметок, что сохранит копию файла в записной книжке. В OneNote можно даже вставлять детали собраний и задач Outlook, а также отправлять электронным сообщением страницу OneNote в Outlook. Можно также создавать или импортировать электронные таблицы Excel и схемы Visio прямо в OneNote и редактировать их информацию непосредственно в заметках. Вставленные файлы показываются в виде значков на странице заметок. Дважды щелкните по любому значку, чтобы открыть файл. 
  
## <a name="equations"></a>Формулы

Недоступна в OneNote для интернета. В классическом приложении OneNote можно кратко записывать математические формулы во время собрания, конференции или занятия, и OneNote сможет моментально посчитать для вас результаты.
  
## <a name="hyperlinks"></a>Гиперссылки

С помощью OneNote для Интернета можно вставить гиперссылку на веб-адрес или применить ссылку на выбранный текст. Вы также можете щелкнуть правой кнопкой мыши вкладку страницы, чтобы скопировать прямую ссылку на эту страницу. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="image-optical-character-recognition-ocr"></a>Распознавание текста изображения (OCR)

Недоступна в OneNote для интернета. Распознавание текста изображения (OCR), которое преобразует текст в виде изображений, например отсканированные документы, в действительные текстовые символы, доступно только в классическом приложении OneNote. 
  
## <a name="ink-viewing"></a>Просмотр рукописных фрагментов

В OneDrive OneNote для веб-отображает ink, но уравнения отображаются в качестве держателей. В SharePoint можно просматривать рукописный ввод, формулы отображаются в виде заполнителей, а фигуры скрыты. Они не могут быть вставлены или изменены в OneNote для Интернета. В OneDrive можно выбирать и удалять фигуры, данные рукописного ввода и формулы-заполнители. В SharePoint можно выбирать и удалять заполнители для этих объектов.
  
## <a name="inking-ink-to-text-and-math-customizable-pens-drawing-tools"></a>Рукописный ввод: рукописное сообщение в виде текста и математических символов, настраиваемые ручки, средства рисования

Недоступна в OneNote для интернета. В классическом приложении OneNote можно плавно рисовать, очищать и редактировать пальцем, пером или мышью. Если вы предпочитаете писать, а не набирать текст, OneNote может преобразовывать рукописный текст. Если вы откроете записную книжку из библиотеки документов SharePoint, все формулы и данные рукописного ввода в записной книжке будут отображаться в виде заполнителей. 
  
## <a name="linked-notes"></a>Связанные заметки

Возможность привязки сведений к файлам недоступна в OneNote для Интернета, а ссылки на файлы скрыты. Однако ссылки сохраняются в заметках, так что вы можете открыть их в классическом приложении OneNote. 
  
## <a name="navigation"></a>Навигация

С помощью OneNote для веб-страницы можно быстро перейти между страницами с помощью кнопок в стиле back и Forward в стиле браузера на панели инструментов быстрого доступа.
  
## <a name="notebook-management-cross-sectionnotebook-page-filing-section-re-order-createdelete-section-groups"></a>Управление записной книжкой. Файловые операции со страницами между разделами и записными книжками, реорганизация разделов, создание и удаление групп разделов

С помощью OneNote для Интернета можно перетаскивать и перетаскивать страницы и разделы.  
Расширенные функции управления, такие как создание, удаление и перезаказ разделов, доступны только в настольном приложении OneNote.
  
## <a name="offline-viewing-and-authoring"></a>Просмотр и редактирование в автономном режиме

Недоступна в OneNote для интернета. Для Office для интернета требуется подключение к Интернету и веб-браузер. Для просмотра и редактирования документа при отключении от Office Web Apps Server или Интернета необходимо установленное на компьютере настольное приложение OneNote.
  
## <a name="outlook-integration-tasks"></a>Интеграция с Outlook (задачи)

OneNote для Интернета не поддерживает команды, которые работают с Microsoft Outlook, такие как страница электронной почты, задачи Outlook или сведения о собраниях.
  
## <a name="paragraph-formatting"></a>Форматирование абзаца

С помощью OneNote для Интернета можно изменить выравнивание абзаца, увеличить или уменьшить отступ от левого поля или изменить направление текста слева направо. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="pictures"></a>Изображения

С помощью OneNote для Интернета можно вставить изображение из файла или из Bing Images. Кроме того, вы можете изменять размер изображений и добавлять замещающий текст. Дополнительные функции работы с изображениями, например вырезка экрана, отсканированные изображения или изображения в Интернете, доступны только в классическом приложении OneNote. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="print"></a>Печать

Заметки можно распечатать в OneNote для интернета.
  
## <a name="proofing-tools"></a>Средства проверки правописания

OneNote для веб-сайта автоматически проверяет правописание по мере написания пользователями и применяет волнистый красный оттенок для опечатки текста. В приложение включены общие действия автозамены, например исправление стандартных ошибок или преобразование знаков в символы. Кроме того, можно задать язык проверки правописания или выключить проверку правописания для выделенного текста. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Управление правами: Применение и использование службы управлением правами на доступ к данным и защиты паролем

OneNote для веб-страницы отображает записные книжки, защищенные с помощью управления правами на информацию (IRM). Однако эти записные книжки нельзя изменить в браузере, и вы не можете создавать записные книжки с защитой IRM в OneNote для Интернета. OneNote для Интернета не может открывать документы, зашифрованные с помощью пароля. Расширенные функции защиты документов, такие как создание ноутбуков с защитой от IRM и применение защиты паролей, доступны только в настольном приложении OneNote.
  
## <a name="search-on-page-within-sections"></a>Поиск (на странице, в рамках разделов)

С помощью OneNote для Интернета можно использовать мгновенный поиск, чтобы вспомнить все, что вы когда-либо создавали или сохранили в OneNote. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="search-by-tag-title-authordate-audio-notes"></a>Поиск по: тегу, названию, автору и дате, звуковым заметкам

Недоступна в OneNote для интернета. Дополнительные функции поиска, например поиск по тегу, названию, автору и дате и звуковым файлам, доступны только в классическом приложении OneNote.
  
## <a name="share"></a>Поделиться

Если вы сохранили записную книжку OneNote в библиотеке документов SharePoint, ваша записная книжка OneNote находится в Интернете. Это значит, что для того, чтобы совместно ее использовать, достаточно отправить ссылку на нее вместо вложения в электронное письмо. Выбрав ссылку, пользователи могут читать заметки в своем веб-браузере. Узнайте больше об [общем доступе к заметкам OneNote в Интернете](https://go.microsoft.com/fwlink/p/?LinkID=272945).
  
## <a name="show-or-hide-authors"></a>Отображение или скрытие авторов

С помощью OneNote для Интернета можно отключить теги автора, которые отображаются при редактировании записной книжки. В результате будут удалены инициалы, которые появляются рядом с новыми заметками на странице. Теги автора включены по умолчанию.
  
## <a name="styles"></a>Стили

С помощью OneNote для Интернета можно легко применить текстовые стили для быстрого форматирования. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="tables"></a>Таблицы

С помощью OneNote для Интернета можно вставить таблицу, изменить текст таблицы и легко изменить основную структуру таблицы, например добавление или удаление строк и столбцов. Расширенные функции работы с таблицами (например, преобразование таблицы в электронную таблицу Excel или заливка ячеек, строки заголовков и сортировка данных внутри ячеек таблицы) доступны в классическом приложении OneNote. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="template-support"></a>Поддержка шаблонов

Недоступна в OneNote для интернета. В классическом приложении OneNote можно использовать шаблон как оформление страницы, которое можно применить к новым страницам в записной книжке, чтобы они имели привлекательный фон, единообразный внешний вид или согласованный макет.
  
## <a name="undo-and-redo"></a>Undo and redo

С помощью клавиатуры можно отменить (Ctrl+ Z) или перезадотировать (Alt + F7) последние действия для каждой страницы, которая редактирована в активной записной книжке в OneNote для Интернета. OneNote для Интернета предоставляет отдельную историю отмены для каждой страницы, отредактируемой в активной записной книжке. Можно отменять неопределенное количество действий на одной странице в течение текущего сеанса редактирования, пока не будет вставлено изображение или внесены изменения другим автором. Перемещение и удаление страниц не может быть отменено. Узнайте больше о [ярлыках OneNote для веб-клавиатуры.](https://go.microsoft.com/fwlink/p/?LinkId=272944)
  
## <a name="video-playback"></a>Воспроизведение видео

Примечания к видео сохраняются в заметках, но не могут быть записаны в OneNote для Интернета. Чтобы воспроизвести видеофайлы, можно загрузить их на свой компьютер. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="view-previous-page-versions"></a>Просмотр предыдущих версий страницы

С помощью OneNote для Интернета можно просматривать и восстанавливать предыдущие версии страницы, в том числе о том, кто ее написал и когда. Изменения по отношению к предыдущим версиям страницы автоматически выделяются. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
