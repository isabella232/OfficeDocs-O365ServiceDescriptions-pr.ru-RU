# <a name="powerpoint-for-the-web"></a>PowerPoint в Интернете

PowerPoint для Интернета (ранее PowerPoint Веб-приложение) расширяет PowerPoint Microsoft для веб-браузера, где можно работать с презентациями непосредственно на веб-сайте, где хранится презентация. Microsoft 365 клиенты с Office для Интернета могут просматривать, создавать и изменять файлы в режиме go.
  
Редактор PowerPoint является веб-компонентом, который создает поверхность редактирования на основе браузера, которая позволяет пользователям работать над документами без потери верности.
  
Если на панели инструментов PowerPoint кнопка Open **in PowerPoint** PowerPoint для Интернета, презентация откроется в настольном приложении PowerPoint (если Microsoft® PowerPoint® 2010, русская версия или позже установлен на компьютере).
  
Узнайте, как скачать и [установить Office с Microsoft 365 для бизнеса на компьютере.](https://support.office.com/article/72977511-dfd1-4d8b-856f-405cfb76839c)
  
## <a name="alignment-bullets-numbers"></a>Alignment, bullets, numbers

С PowerPoint для Интернета вы можете изменить выравнивание абзаца, применить пули или нуминг, а также изменить уровень текста с пулями или номерами. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-basic-transitions-and-animations"></a>Применение основных переходов и анимаций

Выбирайте нужные элементы в коллекции эффектов перехода и анимации. PowerPoint для Интернета поддерживает восемь переходов и 37 анимаций. Дополнительные анимации и переходы, не поддерживаемые PowerPoint для Интернета, сохраняются в презентации и отображаются в слайд-шоу, но их нельзя изменить в PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>Apply rich formatting to text, shapes, and pictures

С PowerPoint для Интернета можно добавить текстовое поле или выбрать из галереи фигур и применить стили, которые определяют эффекты заполнения, контура и тени.

Темы, не поддерживаемые PowerPoint для Интернета, сохраняются в презентации, но возможность изменять темы недоступна в PowerPoint для Интернета. Вы не можете создавать скриншоты в PowerPoint для Интернета, но скриншоты, которые отображаются на презентации в качестве изображений в PowerPoint для Интернета. ## <a name="real-time-co-authoring"></a>Real-time co-authoring

Несколько авторов могут работать одновременно в PowerPoint для Интернета и PowerPoint 2016. Присутствие в режиме реального времени позволяет увидеть, где ваши соавторы работают в презентации, чтобы не создавать конфликтов при редактировании, а также видеть изменения по мере их создания. PowerPoint 2013 поддерживает одновременный редактирование, но нет указания на присутствие, и изменения не могут быть замечены несколькими авторами до тех пор, пока документ не будет сохранен. Дополнительные сведения о совместной работе в режиме реального времени в PowerPoint см. в PowerPoint [презентации.](https://go.microsoft.com/fwlink/?linkid=844014)

## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Управление правами. Применение и использование службы управлением правами на доступ к данным и защиты паролем

Недоступна в PowerPoint для Интернета. PowerPoint для Интернета отображает презентации, защищенные с помощью управления правами на информацию (IRM). PowerPoint для Интернета не могут открывать презентации, подписанные в цифровом виде или зашифрованные с помощью пароля.

## <a name="view-and-add-comments"></a>View and add comments

Если файлы хранятся в SharePoint Или OneDrive для бизнеса, вы можете добавить, изменить или удалить комментарии. 
  
Если файлы хранятся в OneDrive, можно добавить, изменить или удалить комментарии во время просмотра редактирования, а также просматривать и обновлять комментарии во время просмотра чтения.
  
## <a name="view-and-edit-slide-notes"></a>View and edit slide notes

С PowerPoint для Интернета можно отображать или скрывать заметки для каждого слайда. Также можно добавить заметки в режиме редактирования.
  
## <a name="wysiwyg-viewing"></a>Просмотр в режиме WYSIWYG

Редактируйте презентацию практически в том же виде, что и при печати или показе готового продукта.
