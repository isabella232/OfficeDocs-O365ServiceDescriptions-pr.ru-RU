# <a name="sharepoint-service-description"></a>Описание службы SharePoint

SharePoint в Microsoft 365 помогает организациям обмениваться контентом, знаниями и приложениями для расширения возможностей совместной работы, быстрого поиска информации и бесперебойной совместной работы в организации. Краткий обзор см. [в SharePoint?](https://support.office.com/article/97b915e6-651b-43b2-827d-fb25777f446f)

## <a name="available-plans"></a>Доступные планы

Microsoft 365 доступны в различных планах, чтобы наилучшим образом соответствовать потребностям вашей организации. Подробные сведения о планах подписки, которые позволяют пользователям SharePoint, см. в полной [таблице сравнения подписки](https://go.microsoft.com/fwlink/?linkid=2139145). Чтобы получить SharePoint, пользователям в организации необходимо получить лицензию, которая включает SharePoint. План Firstline не продается в качестве автономных предложений, только в рамках [Office 365 F3](https://www.microsoft.com/microsoft-365/business/office-365-f1)[, Microsoft 365 F1](https://www.microsoft.com/microsoft-365/enterprise/f1?activetab=pivot:overviewtab) [или Microsoft 365 F3](https://www.microsoft.com/microsoft-365/enterprise/f3?activetab=pivot:overviewtab).

## <a name="learn-more"></a>Дополнительные сведения

### <a name="sharepoint-resources"></a>SharePoint Ресурсы

**OneDrive:** OneDrive в Microsoft 365 это хранилище для отдельных пользователей в организации. Это&#39;место, где люди могут хранить, синхронизировать и делиться своими файлами работы на нескольких устройствах с легкостью и безопасностью. См. [OneDrive описание](/office365/servicedescriptions/onedrive-for-business-service-description) службы, чтобы узнать, OneDrive хранения поставляется с каждым планом. OneDrive включена в SharePoint и также может быть приобретена как автономный план.

**Права гостей:** Если вы покупаете план и создаете сайт, на который используются корпоративные функции, гостям, которые вы приглашаете, пользуется правами на использование и/или просмотр корпоративных функций на сайте. Хотя вы можете приглашать гостей выполнять полный спектр действий на сайте, они&#39;не имеют таких же возможностей, как лицензированный пользователь в вашей организации. Например, если ваш план включает настольные версии Office приложений, гости могут&#39;установить их на своих компьютерах, если вы не назначите им лицензию.

SharePoint для правительства **США:** SharePoint правительственных сред [США, чтобы](/office365/servicedescriptions/office-365-platform-service-description/office-365-us-government/sharepoint) узнать больше о различиях функций для государственных облачных клиентов США.

**SharePoint:** установите [SharePoint для iOS, Android или Windows](https://products.office.com/sharepoint/mobile-app), чтобы работать с SharePoint сайтов на мобильном устройстве.

**Sync App:** Чтобы синхронизировать SharePoint на компьютере, пользователи могут использовать [приложение приложение синхронизации OneDrive для](https://support.office.com/article/sync-sharepoint-and-teams-files-with-the-onedrive-sync-app-6de9ede8-5b6e-4503-80b2-6190f3354a88) Windows или Mac. В [описании OneDrive](/office365/servicedescriptions/onedrive-for-business-service-description) службы см. описание доступности функций в приложении синхронизации.
