# <a name="office-applications-service-description"></a>Описание службы приложений Office

Microsoft 365 (прежнее название — Office) — это служба подписки, которая предоставляет последнюю версию уже знакомых Microsoft 365 приложений, таких как Word, Excel и PowerPoint.

Хотя это облачная служба, Microsoft 365 приложения не выполняются в облаке. Вместо этого пользователи загружают приложения с портала Microsoft 365 и устанавливают их на локальные компьютеры, доступные в операционных системах Windows, Mac, IOS и Android. Корпорация Майкрософт использует технологию "нажми и запускай", чтобы быстро и просто Microsoft 365 приложений. Технология "нажми и запускай" использует технологию виртуализации для запуска приложений Microsoft 365 в автономной среде на локальном компьютере, что позволяет пользователям запускать приложения Microsoft 365 параллельно с более ранними версиями Office. Microsoft 365 приложения доступны в 32-разрядных\* и 64-разрядных выпусках.

Подробные сведения о планах корпоративных подписок, которые позволяют пользователям Microsoft 365 описание службы приложений, см. в полной таблице сравнения [подписок](https://aka.ms/M365EnterprisePlans).
## <a name="available-plans"></a>Доступные планы
Подробные сведения о планах подписок, которые позволяют пользователям Microsoft 365 приложениях, см. в Microsoft 365 корпоративный [планах](https://www.microsoft.com/microsoft-365/enterprise/compare-office-365-plans).

## <a name="microsoft-365-apps-availability-in-plans-for-mac"></a>Microsoft 365 приложений в планах для Mac
Следующие Office доступны для Приложения Microsoft 365 для Enterprise для Mac и Office LTSC 2021 для Mac: [Microsoft Word](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-word), [Microsoft Excel](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-excel), [Microsoft PowerPoint ](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-powerpoint), [Microsoft OneNote](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-onenote), [Microsoft Outlook](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-outlook) [и Microsoft Teams](/office365/servicedescriptions/teams-service-description).
Office для Mac поддерживается в трех последних версиях Mac OS. По мере того как новые основные версии Mac OS будут общедоступными, корпорация Майкрософт перестанет поддерживать старую версию и поддерживать последнюю и предыдущие две версии Mac OS. Функциональные возможности продукта и доступность функций могут отличаться в старых системах. Дополнительные сведения см[. в Microsoft 365 и Microsoft 365 приложениях](https://products.office.com/office-system-requirements).
Следующие Office приложения недоступны для Приложения Microsoft 365 для Enterprise для Mac и Office LTSC 2021 для Mac: [Microsoft Publisher](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-publisher), [Microsoft Access](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-access) и [Microsoft InfoPath](/office365/servicedescriptions/office-applications-service-description/office-applications#microsoft-infopath).
## <a name="microsoft-365-apps-and-feature-availability-in-government-plans"></a>Microsoft 365 приложений и функций в планах для государственных организаций
