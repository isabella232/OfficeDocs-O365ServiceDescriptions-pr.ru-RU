# <a name="microsoft-365-and-office-365-platform-service-description"></a>Описание служб платформы Microsoft 365 и Office 365

Microsoft 365 и Office 365 доступны в различных планах в соответствии с потребностями вашей организации. Если вы ищете различия между планами, включая планы Exchange Online, в этой статье показано, какие функции включены в каждый из них.
  
Microsoft 365 и Office 365 обеспечивают эффективность облачной производительности для предприятий любого размера, помогая экономить время, деньги и освободить ресурсы. Планы Microsoft 365 и Office 365 объединяют знакомый набор классических приложений Microsoft Office с облачными версиями служб связи и совместной работы нового поколения Майкрософт, включая Microsoft Exchange Online, Microsoft Office SharePoint Online, Office для Интернета и Microsoft Skype для бизнеса Online, чтобы помочь пользователям эффективно работать практически из любого места в Интернете.
  
Подробные сведения о планах подписок, которые позволяют пользователям использовать Microsoft 365 и Office 365, см. в полной таблице сравнения [подписок](https://go.microsoft.com/fwlink/?linkid=2139145). Более подробный список функций см. в соответствующем описании службы Microsoft 365 [и Office 365 службы](/office365/servicedescriptions/office-365-service-descriptions-technet-library).
  
- Сведения о Office 365 для образования планах см. на Office 365 для образования [планах](https://aka.ms/EDU-Plan-Comparison).

- Если вас интересуют планы Office 365 US Government, начните со страницы [Office 365 для государственных организаций США](office-365-us-government/office-365-us-government.md). 
    
- Если вас интересует Office 365 под управлением 21Vianet, начните со страницы [Служба Office 365, которой управляет 21Vianet](office-365-operated-by-21vianet.md). 
    
## <a name="office-365-plans"></a>Планы Office 365

Служба Office 365 доступна в разных планах для того, чтобы наилучшим образом удовлетворить потребности вашей организации. Сведения о различных планах, включая параметры автономного плана и сведения о переходе от одного плана к другому, см. Office 365 [плана](office-365-plan-options.md).
  
Дополнительные сведения о корпоративном лицензировании см [. в разделе "Условия лицензирования и документация"](https://go.microsoft.com/fwlink/?linkid=848696).
  
## <a name="microsoft-365-and-office-365-plans-and-pricing-for-nonprofits"></a>Microsoft 365 и Office 365 и цены для некоммерческих организаций

Для организаций, которые имеют право на Microsoft 365 и Office 365 для некоммерческих организаций, эти планы включают те же функции, что и соответствующие бизнес-планы (например, Office 365 E5 для некоммерческих организаций включает те же функции, что и Office 365 E5). Единственное отличие между эквивалентными планами — цена. Дополнительные сведения о планах Microsoft 365 и Office 365 для некоммерческих организаций см. в статье [Microsoft 365 и Office 365](https://go.microsoft.com/fwlink/?LinkID=627221) предложения для некоммерческих организаций и решения для повышения производительности для некоммерческих [организаций](https://go.microsoft.com/fwlink/?LinkID=786641).

## <a name="multi-geo-capabilities"></a>Возможности с поддержкой нескольких регионов 

Поддержка нескольких регионов позволяет одной организации охватывать несколько регионов Office 365 центров обработки данных и позволяет хранить данные для Exchange Online и OneDrive для бизнеса, неактивных данных для каждого пользователя в выбранном географическом регионе. 
  
Доступные географические регионы: Азиатско-Тихоокеанский регион, Австралия, Канада, Европейский союз, Индия, Япония, Корея, Соединенное Королевство, США. 
  
Дополнительные сведения см. [в статье "Знакомство с поддержкой нескольких регионов в Office 365](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Introducing-Multi-Geo-in-Office-365/ba-p/107016)".
