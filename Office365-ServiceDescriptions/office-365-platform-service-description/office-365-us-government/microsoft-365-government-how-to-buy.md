# <a name="microsoft-365-government---how-to-buy"></a>Microsoft 365 Правительство - как купить

В ответ на уникальные и меняющиеся потребности государственного сектора Соединенных Штатов корпорация Майкрософт создала Microsoft 365 условия для клиентов государственного сектора США. В этом описании сервиса содержится обзор сред и более подробная информация о том, как купить. Рекомендуется прочитать это описание сервиса наряду с другими [Microsoft 365 и Office 365 сервисом.](../../office-365-service-descriptions-technet-library.md)
  
## <a name="whats-microsoft-365-government"></a>Что такое Microsoft 365 правительство?

Microsoft 365 Правительство — это набор возможностей облачного программного обеспечения для обеспечения производительности, безопасности и мобильности, разработанный специально для правительственных учреждений и подрядчиков США, спонсируемых для хранения контролируемой, несекретной информации. Поставляемое через уникальные среды, которые отвечают самым строгим требованиям соответствия, Microsoft 365 Government является облачным предложением для государственных клиентов США, которое соответствует как можно более тесно функциям и возможностям коммерческих облачных корпоративных предложений Microsoft.
  
## <a name="whats-the-value-of-microsoft-365-government"></a>Какова ценность Microsoft 365?

Microsoft 365 Правительство помогает руководителям и работникам правительства США быть продуктивными в любом месте, в любое время, позволяя производительности мобильных устройств и обеспечить сотрудничество с соблюдением, помогая учреждениям эффективно управлять нормативными требованиями к соблюдению данных. ## <a name="what-makes-microsoft-365-government-different-from-microsoft-365-for-enterprise-offerings"></a>Чем Microsoft 365 правительство отличается от Microsoft 365 для корпоративных предложений?

Microsoft 365 Правительство уникально предназначено для удовлетворения требований государственных учреждений США в четырех ключевых областях:

**Эксклюзивное сообщество.** Правительственные центры обработки данных доступны только правительственным учреждениям или ведомствам или коммерческим компаниям, которые были уполномочены вести и обрабатывать контролируемую информацию от имени правительства США.

**Скрининг персонала.** Доступ к содержимому клиентов вашей организации ограничен скринингом сотрудников корпорации Майкрософт, которые являются гражданами США, и прошел необходимые проверки.
Сторонние аудиты. Вся инфраструктура проверяется сертифицированным сторонним аудитором, который предоставляет отчет об оценке безопасности или письмо-заверение, используемое федеральными ведомствами для выдачи разрешения на эксплуатацию (АТО). 

**Хранение содержимого.** Данные хранятся на территории континентальной части Соединенных Штатов с соответствующей инфраструктурой. В следующей таблице кратко излагаются Microsoft 365 для правительства окружающей среды и обязательств.<br><br>

| Предложения | Обязательства |
|:----------|:----------------------------------|
|Microsoft 365 Правительственные GCC  <br/> |FedRamp High, DFARS, CJIS, IRS 1075, DISA SRG L2 <br/> |
|Microsoft 365 Правительство GCC высокий  <br/> |FedRamp High, ITAR, DFARS, DISA SRG L4 <br/> |

## <a name="microsoft-365-government-eligibility-and-validation"></a>Microsoft 365 Право правительства и проверка

Microsoft 365 Правительство доступно соответствующим государственным заказчикам и неправительственным организациям, спонсируемым для хекономя, для хекономя или обработки контролируемой информации. Критерии приемлемости для этих двух категорий клиентов согласованы в облаке правительства Майкрософт.

Как GCC, так и GCC High доступны любому клиенту, имеюму право на участие в программе Microsoft Government Cloud, а среда Министерства обороны используется исключительно для Министерства обороны США. Доступность услуг и цена различаются, и GCC остается героем предложения для всех клиентов, которые не держат FedRAMP High или DOD контролируемой несекретной информации (CUI).

Подходящий государственный заказчик определяется следующим образом:

- Федеральное агентство, определяемое как бюро, офис, агентство, департамент или другое учреждение правительства США
- Состояние/местное образование, определяемое как одно из следующих:
    - Любое учреждение штата или местного самоуправления в США.
    - Любой округ США, район, содружество, город, муниципалитет, город, поселок, район специального назначения или другой подобный тип государственной инструментальности, установленный законами штата заказчика и расположенный в пределах юрисдикции и географических границ государства клиента
    - Округ Колумбия, Содружество Пуэрто-Рико, Гуам, Американское Самоа, Виргинские острова Соединенных Штатов и Северные Марианские острова
- Племенное образование, определяемое как признанное на федеральном уровне племенное образование, имею которое имеет право на финансирование и услуги от Министерства внутренних дел США в силу своего статуса индейского племени, или, на Аляске, родной деревни или Региональной корпорации коренных жителей Аляски

Неправительственные организации, которые имеют следующие типы данных и могут предоставить перечисленные доказательства, имеют право на процесс проверки облачных технологий правительства Майкрософт:

Коммерческое частное предприятие с данными, на которые распространяются правила. Принятые типы правительственных данных включают:
- Международный трафик в области вооружений (ITAR)
- Контролируемая несекретная информация (CUI)
- Министерство обороны (DOD) Несекретная контролируемая ядерная информация (UCNI)
- Министерство энергетики (DOE) UCNI
- Информация об уголовном правосудии (CJI)
- Данные уровня воздействия Министерства обороны
- Другие типы данных, которые требуют Microsoft 365 правительства

Международная коммерческая организация может претендовать, хотя регулируемые данные могут потребоваться для покупки услуги через их дочернюю компанию в США.

Для доступа к предложениям правительства США потребуется подтверждение членства в одной из перечисленных выше групп Microsoft 365 предложения.
