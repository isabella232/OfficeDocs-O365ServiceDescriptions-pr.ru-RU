# <a name="microsoft-365-suite-features"></a>Компоненты набора Microsoft 365

Перечисленные в этом разделе функции — это коллекция веб-средств, которые помогают вам сотрудничать и безопасно общаться на многих устройствах и нескольких Office 365 службах. Эти функции помогают налаживать новые связи в вашей организации, объединяя создание цифрового контента, хранилище и управление со средствами социальных сетей для поиска информации и обмена нею между пользователями.
  
## <a name="delve"></a>Delve

Delve позволяет пользователям искать и обнаруживать контент во всех Microsoft 365 на основе персонализированных данных. Delve — это первое решение на платформе Office Graph. Чтобы узнать больше о Delve, перейдите в [What is Delve?.](https://go.microsoft.com/fwlink/?LinkId=511463)

## <a name="microsoft-briefing-email"></a>Электронная почта Microsoft Briefing

Электронная почта брифинга помогает пользователям использовать каждый день, помогая им:

* Подготовка к собраниям с указанием связанных задач или соответствующих документов.
* Управление задачами, идентифицированные по электронной почте, которые могут быть отложены.
* Сделайте большую часть времени фокуса, забронировав время в календаре и отключение уведомлений.

Электронная почта брифинга предоставляется автоматически всем пользователям Exchange Online. Дополнительные сведения см. в [обзоре электронной почты брифинга.](/Briefing/be-overview)

## <a name="microsoft-power-automate"></a>Microsoft Power Automate

Microsoft Power Automate позволяет автоматизировать рабочий процесс в приложениях. Flow позволяет подключать оповещения об электронной почте и мгновенных сообщениях, синхронизировать файлы между приложениями, копировать файлы из одной службы в другую, собирать данные из одного приложения и хранить их в другом и т. д. Для начала работы доступны шаблоны. Чтобы узнать больше о Flow возможностях и их использовании, перейдите на [веб-сайт Power Automate.](https://preview.flow.microsoft.com/)
  
## <a name="microsoft-forms"></a>Microsoft Forms

Microsoft Forms позволяет быстро и легко создавать настраиваемые викторины, опросы, анкеты, регистрации и другие. После создания викторины или какой-либо формы можно предложить другим пользователям ответить на вопросы. При этом они могут использовать любой браузер, в том числе на мобильном устройстве. Данные форм, такие как результаты викторины, можно легко экспортировать в Excel для дополнительного анализа или классификации. Дополнительные дополнительные информации см. [в веб-сайте What is Microsoft Forms?.](https://support.office.com/forms)
  
## <a name="microsoft-graph-api"></a>API Microsoft Graph

Используйте API microsoft Graph, чтобы быстро создавать приложения, которые подключаются к богатым ресурсам, отношениям и интеллекту, все это через одну конечную точку: https://graph.microsoft.com . Например, можно создать настраиваемые панели мониторинга, рабочий процесс и приложения, которые будут получать данные для организации. Дополнительные сведения см. в [документации microsoft Graph.](/graph/overview)
  
## <a name="microsoft-lists"></a>Microsoft Lists

Microsoft Lists — это Microsoft 365 приложение, которое помогает отслеживать сведения и организовывать работу. Списки просты, умны и гибки, поэтому вы можете оставаться на вершине того, что наиболее важно для вашей команды. Отслеживайте проблемы, активы, процедуры, контакты, инвентаризацию и другие параметры с помощью настраиваемых представлений и интеллектуальных правил, чтобы все были уведомлены и синхронизированы. С помощью готовых шаблонов можно быстро запустить списки в Интернете, в нашем новом мобильном приложении или Microsoft Teams. Так как это часть Microsoft 365, вы можете полагаться на безопасность и соответствие требованиям, готовым к корпоративному обеспечению.

## <a name="microsoft-planner"></a>Планировщик (Майкрософт)

"Планировщик" (Майкрософт)  это инструмент, предоставляющий пользователям наглядный способ организации командной работы. Команды могут создавать планы, формировать и назначать задачи, делиться файлами, обсуждать ход выполнения работы, устанавливать сроки выполнения и обновлять статус. Кроме того, "Планировщик" позволяет связывать документы с конкретными задачами, совместно редактировать их и открывать беседы по задачам. Дополнительные сведения см. в статье [Знакомство с приложением "Планировщик" (Майкрософт)](https://go.microsoft.com/fwlink/?LinkID=718016&amp;clcid=0x4809). Это приложение пока не доступно в [Office 365 в Германии](office-365-germany.md).
  
## <a name="microsoft-power-apps"></a>Microsoft Power Apps

Microsoft Power Apps — это корпоративная служба, которая помогает быстро создавать, интегрировать и обмениваться приложениями, которые работают на любом устройстве. Power Apps позволяет использовать встроенные подключения или подключенные вашей компанией для подключения приложения к облачным и локальной службам, включая базы данных Microsoft 365 и Office 365, Dynamics CRM, OneDrive, SharePoint, SQL Server, базы данных Oracle и другие. Вы можете легко обмениваться Power Apps коллегами, введя адрес электронной почты. Чтобы узнать больше, перейдите на [веб-сайт](https://powerapps.microsoft.com/)Microsoft Power Apps.
  
## <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream — это корпоративная видео-служба, которую люди в вашей организации могут использовать для безопасной загрузки, просмотра и обмена видео. Вы можете поделиться записями занятий, собраний, презентаций, учебных семинаров и другими видеофайлами, которые будут полезны для совместной работы вашей команды. Microsoft Stream также позволяет легко делиться комментариями о видео и тегами кодов времени в комментариях и описаниях для ссылки на определенные точки в видео.
  
## <a name="microsoft-sway"></a>Microsoft Sway

Microsoft Sway — это профессиональное приложение для создания цифровых историй для бизнеса, которое помогает вам и вашим коллегам выражать идеи с помощью интерактивного веб-холста. Встроенный проектный движок Microsoft Sway позволяет создавать профессиональные, визуально привлекательные отчеты, презентации и другие отчеты без необходимости широкого форматирования или дополнительного обучения. Вы также можете изменить результаты, чтобы получить уникальный внешний вид и почувствовать, что вы хотите. Microsoft Sway делает ваше создание отличным в любом браузере на любом экране, и его можно делиться с коллегами и клиентами, отправив ссылку. Microsoft Sway помогает вам находить и стягивать все виды контента, не выходя из приложения, поэтому вы можете перетаскивать изображения, текст, видео и диаграммы прямо на холст. Дополнительные информации см. [в см. в .Sway.](https://sway.com) Microsoft Sway еще не доступна в Office 365 [Германии](office-365-germany.md).

## <a name="windows-update-for-business-deployment-service"></a>Служба развертывания Windows Update для бизнеса 

Служба Windows обновления для бизнеса обеспечивает контроль над утверждением, планированием и защитой контента, доставленного Windows Update. Он предназначен для работы с существующими политиками Windows для бизнеса, чтобы обеспечить полный контроль над отдельными обновлениями. Возможности, предоставляемые службой развертывания, включают:

* Расписание развертывания обновлений функций для начала в определенную дату
* Поэтапное развертывание в течение нескольких дней или недель с использованием насыщенных выражений
* Обход предварительно настроенных Windows для бизнес-политик, чтобы немедленно развернуть обновление безопасности в организации
* Обеспечение охвата оборудования и программного обеспечения в организации с помощью развертывания, адаптированного к вашему уникальному населению устройств.
* Автоматическое определение и приостановка развертывания на устройствах, на которые, скорее всего, повлияет [удержание защиты](/windows/deployment/update/safeguard-holds)

Эти возможности доступны с помощью API Graph Microsoft и интеграции с Microsoft Endpoint Manager. Дополнительные сведения см. [в Windows Обзор службы](https://aka.ms/wufb/deployment-service)развертывания для бизнеса.
