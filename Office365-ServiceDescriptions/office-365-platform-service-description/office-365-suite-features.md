# <a name="microsoft-365-suite-features"></a>Компоненты набора Microsoft 365

Перечисленные в этом разделе функции — это коллекция веб-средств, которые помогают вам сотрудничать и безопасно общаться на многих устройствах и нескольких службах Office 365. Эти функции помогают налаживать новые связи в вашей организации, объединяя создание цифрового контента, хранилище и управление со средствами социальных сетей для поиска информации и обмена нею между пользователями.
  
## <a name="delve"></a>Delve

Delve позволяет пользователям искать и открывать контент в Microsoft 365 на основе персонализированных данных. Delve — это первое решение на платформе Office Graph. Чтобы узнать больше о Delve, перейдите в [What is Delve?.](https://go.microsoft.com/fwlink/?LinkId=511463)

## <a name="microsoft-briefing-email"></a>Электронная почта Microsoft Briefing

Электронная почта брифинга помогает пользователям использовать каждый день, помогая им:

* Подготовка к собраниям с указанием связанных задач или соответствующих документов.
* Управление задачами, идентифицированные по электронной почте, которые могут быть отложены.
* Сделайте большую часть времени фокуса, забронировав время в календаре и отключение уведомлений.

Электронная почта брифинга предоставляется автоматически всем пользователям Exchange Online. Дополнительные сведения см. в [обзоре электронной почты брифинга.](/Briefing/be-overview)

## <a name="microsoft-power-automate"></a>Microsoft Power Automate

Microsoft Power Automate позволяет автоматизировать рабочий процесс в приложениях. Flow позволяет подключать оповещения об электронной почте и мгновенных сообщениях, синхронизировать файлы между приложениями, копировать файлы из одной службы в другую, собирать данные из одного приложения и хранить их в другом и т. д. Для начала работы доступны шаблоны. Чтобы узнать больше о возможностях Flow и их использовании, перейдите на веб-сайт [Power Automate.](https://preview.flow.microsoft.com/)
  
## <a name="microsoft-forms"></a>Microsoft Forms

Microsoft Forms позволяет быстро и легко создавать настраиваемые викторины, опросы, анкеты, регистрации и другие. После создания викторины или какой-либо формы можно предложить другим пользователям ответить на вопросы. При этом они могут использовать любой браузер, в том числе на мобильном устройстве. Данные форм, такие как результаты викторины, можно легко экспортировать в Excel для дополнительного анализа или классификации. Дополнительные дополнительные информации см. [в веб-сайте What is Microsoft Forms?.](https://support.office.com/forms)
  
## <a name="microsoft-graph-api"></a>API Microsoft Graph

Используйте API Microsoft Graph для быстрой сборки приложений, которые подключаются к богатым ресурсам, отношениям и интеллекту с помощью одной конечной точки: https://graph.microsoft.com . Например, можно создать настраиваемые панели мониторинга, рабочий процесс и приложения, которые будут получать данные для организации. Дополнительные сведения см. в [документации Microsoft Graph.](/graph/overview)
  
## <a name="microsoft-lists"></a>Microsoft Lists

Microsoft Lists — это приложение Microsoft 365, которое помогает отслеживать сведения и организовывать работу. Списки просты, умны и гибки, поэтому вы можете оставаться на вершине того, что наиболее важно для вашей команды. Отслеживайте проблемы, активы, процедуры, контакты, инвентаризацию и другие параметры с помощью настраиваемых представлений и интеллектуальных правил, чтобы все были уведомлены и синхронизированы. С помощью готовых шаблонов можно быстро запускать списки в Интернете, в новом мобильном приложении или в Microsoft Teams. Как часть Microsoft 365, вы можете полагаться на корпоративную безопасность и соответствие требованиям.

## <a name="microsoft-planner"></a>Планировщик (Майкрософт)

"Планировщик" (Майкрософт)  это инструмент, предоставляющий пользователям наглядный способ организации командной работы. Команды могут создавать планы, формировать и назначать задачи, делиться файлами, обсуждать ход выполнения работы, устанавливать сроки выполнения и обновлять статус. Кроме того, "Планировщик" позволяет связывать документы с конкретными задачами, совместно редактировать их и открывать беседы по задачам. Дополнительные сведения см. в статье [Знакомство с приложением "Планировщик" (Майкрософт)](https://go.microsoft.com/fwlink/?LinkID=718016&amp;clcid=0x4809). Это приложение пока не доступно в [Office 365 в Германии](office-365-germany.md).
  
## <a name="microsoft-power-apps"></a>Microsoft Power Apps

Microsoft Power Apps — это корпоративная служба, которая помогает быстро создавать, интегрировать и обмениваться приложениями, которые работают на любом устройстве. Power Apps позволяет использовать встроенные подключения или встроенные подключения, построенные вашей компанией, для подключения приложения к облачным и локальной службам, включая базы данных Microsoft 365 и Office 365, Dynamics CRM, OneDrive, SharePoint, SQL Server, базы данных Oracle и другие. Вы можете легко поделиться своими приложениями power с коллегами, введя адрес электронной почты. Чтобы узнать больше, перейдите на веб-сайт Microsoft [Power Apps.](https://powerapps.microsoft.com/)
  
## <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream — это корпоративная видео-служба, которую люди в вашей организации могут использовать для безопасной загрузки, просмотра и обмена видео. Вы можете поделиться записями занятий, собраний, презентаций, учебных семинаров и другими видеофайлами, которые будут полезны для совместной работы вашей команды. Microsoft Stream также позволяет легко делиться комментариями о видео и тегами кодов времени в комментариях и описаниях для ссылки на определенные точки в видео.
  
## <a name="microsoft-sway"></a>Microsoft Sway

Microsoft Sway — это профессиональное приложение для создания цифровых историй для бизнеса, которое помогает вам и вашим коллегам выражать идеи с помощью интерактивного веб-холста. Встроенный проектный движок Microsoft Sway позволяет создавать профессиональные, визуально привлекательные отчеты, презентации и другие отчеты без необходимости широкого форматирования или дополнительного обучения. Вы также можете изменить результаты, чтобы получить уникальный внешний вид и почувствовать, что вы хотите. Microsoft Sway делает ваше создание отличным в любом браузере на любом экране, и его можно делиться с коллегами и клиентами, отправив ссылку. Microsoft Sway помогает вам находить и стягивать все виды контента, не выходя из приложения, поэтому вы можете перетаскивать изображения, текст, видео и диаграммы прямо на холст. Дополнительные информации см. [в см. в .Sway.](https://sway.com) Microsoft Sway еще не доступна в [Office 365 Германии](office-365-germany.md).

Microsoft 365 Apps для предприятия — это всегда доступный в качестве подписки набор настольных приложений, которые вы уже знаете (в том числе Word, PowerPoint, Excel, Outlook и Teams). Приложения Microsoft 365 для предприятия включают эксклюзивные интеллектуальные возможности, такие как [**Ideas in Excel,**](https://go.microsoft.com/fwlink/p/?linkid=2109915) [**Researcher in Word,**](https://go.microsoft.com/fwlink/p/?linkid=2109916)совместная работа в режиме реального [**времени,**](https://go.microsoft.com/fwlink/p/?linkid=2109917)Teams как центр совместной работы и расширенные функции безопасности. В качестве пользовательской лицензии можно развернуть Office на пяти компьютерах или mac, пяти планшетах и пяти мобильных устройствах. С помощью 1 ТБ облачного хранилища OneDrive вы можете получать доступ к файлам из любой точки мира и сотрудничать с ними.

## <a name="microsoft-365-groups"></a>Группы Microsoft 365

Группы в Microsoft 365 соединяют пользователей с коллегами, информацией и приложениями, которые им необходимы для совместной работы. Группы по умолчанию открыты для повышения доступности обнаружения и общего доступа, но пользователи также могут создавать частные группы для конфиденциального контента. Дополнительные информацию о группах см. в [группе Groups в Microsoft 365.](https://support.office.com/Article/Find-help-about-Groups-in-Office-365-7a9b321f-b76a-4d53-b98b-a2b0b7946de1) Сведения о ограничениях групп см. в "Как управлять группами" в "Сведения о группах [Microsoft 365".](https://go.microsoft.com/fwlink/?linkid=846714)

## <a name="microsoft-kaizala"></a>Microsoft Kaizala

Microsoft Kaizala — это простое и безопасное приложение для управления мобильными работами, которое обеспечивает уникальные возможности Microsoft 365 с возможностью подключения и привлечения людей как внутри, так и за пределами каталога организации, включая контрактников, поставщиков, партнеров, поставщиков, клиентов и граждан с помощью больших и гибких типов [групп.](https://techcommunity.microsoft.com/t5/Microsoft-Kaizala-Blog/Model-your-communication-needs-using-Kaizala/ba-p/299710) Модель открытого каталога Kaizala позволяет легко использовать идентификатор на основе номеров телефонов, а также простой пользовательский интерфейс для обмена сообщениями и управления работой. Мобильное приложение Kaizala позволяет пользователям отправлять мгновенные сообщения и вложения, такие как фотографии, документы, видео, аудиофайлы и другие, а также использовать карточки действий для опросов, опросов, заданий, объявлений, тренингов и других. [Kaizala работает](https://products.office.com/en/business/microsoft-kaizala) на iPhone и Android. Чтобы узнать больше, перейдите на [веб-сайт Kaizala](https://products.office.com/en/business/microsoft-kaizala).

Office 365 Video — это безопасное место для всей компании, где можно загружать, делиться и открывать видео на устройствах с питанием от служб Microsoft Azure Media Services. Видео Office 365 помогает поддерживать вовлечение и информированное внимание зрителей, широкое общение с идеями и унисон для организаций. ## <a name="onenote-class-notebook"></a>Записная книжка OneNote для занятий

Ноутбук класса является частью OneNote и доступен для Office 365 Education и Office 365 E5 Education. Каждый блокнот включает библиотеку контента для преподавателей для обмена контентом, пространство совместной работы, где преподаватели и студенты могут работать вместе, а также частный блокнот для каждого учащегося. После создания записной книжки класса преподаватели и учащиеся могут использовать приложение OneNote для доступа к нему с любого устройства.

## <a name="to-do"></a>To Do

Microsoft To Do — это приложение для управления задачами, которое позволяет клиентам планировать свой день и управлять своими задачами — на работе и в жизни — с любого устройства.

С помощью Microsoft To Do клиенты могут:

* Синхронизация задач в Outlook, Teams, Planner и других
* Оставайтесь в центре внимания с My Day, персонализированным ежедневным планировщиком с предложенными задачами
* Получите их списки в любом месте, на любом устройстве или платформе
* Совместное выполнение списков и назначение задач коллегам, друзьям и родственникам
