# <a name="microsoft-365-suite-features"></a>Компоненты набора Microsoft 365

Функции, перечисленные в этом разделе, — это набор веб-средств, которые помогают безопасно взаимодействовать и взаимодействовать на разных устройствах и Office 365 службах. Эти функции помогают налаживать новые связи в вашей организации, объединяя создание цифрового контента, хранилище и управление со средствами социальных сетей для поиска информации и обмена нею между пользователями.
  
## <a name="microsoft-365-apps-for-enterprise"></a>Приложения Microsoft 365 для предприятий

Приложения Microsoft 365 для предприятий — это всегда актуальный набор классических приложений, которые вы уже знаете (включая Word, PowerPoint, Excel, Outlook и Teams), доступных в виде подписки. Приложения Microsoft 365 для предприятий включает в себя эксклюзивные интеллектуальные возможности, такие как [**идеи в Excel, Исследователь**](https://go.microsoft.com/fwlink/p/?linkid=2109915) в [**Word**](https://go.microsoft.com/fwlink/p/?linkid=2109916)[**,**](https://go.microsoft.com/fwlink/p/?linkid=2109917) совместная работа в режиме реального времени, Teams как центр командной работы и расширенные функции безопасности. В качестве пользовательской лицензии вы можете развертывать Office на пяти компьютерах или компьютерах Mac, пяти планшетах и пяти мобильных устройствах. Используя 1 ТБ OneDrive облачного хранилища, вы можете получать доступ к файлам и совместно работать с ними из любого места.

Дополнительные сведения см. в указанных ниже статьях.

* [Страница "Обзор"](https://www.microsoft.com/microsoft-365/business/microsoft-365-apps-for-enterprise)
* [Страница продукта (для покупки)](https://www.microsoft.com/microsoft-365/business/microsoft-365-apps-for-enterprise-product)

## <a name="microsoft-365-groups"></a>Группы Microsoft 365

Группы в Microsoft 365 подключают пользователей к коллегам, сведениям и приложениям, которые им нужны для более совместной работы. Группы открыты по умолчанию для улучшения возможности обнаружения и совместного использования, но пользователи также могут создавать частные группы для конфиденциального содержимого. Дополнительные сведения о группах см. в [Microsoft 365](https://support.office.com/Article/Find-help-about-Groups-in-Office-365-7a9b321f-b76a-4d53-b98b-a2b0b7946de1). Дополнительные сведения об ограничениях групп см. в разделе "Разделы справки управление моими группами" в разделе "Microsoft 365 [групп"](https://go.microsoft.com/fwlink/?linkid=846714).

## <a name="microsoft-briefing-email"></a>Электронная почта microsoft Briefing

Сообщение электронной почты с кратким сообщением помогает пользователям эффективно использовать каждый день, помогая им:

* Подготовка к собраниям путем отображения связанных задач или соответствующих документов.
* Управление задачами, обнаруженными по электронной почте, которые могут быть ожидающими.
* Максимально эффективное время фокусировки за счет резервирования времени в календаре и уведомлений о отключении уведомлений.

Сообщение электронной почты с кратким сообщением автоматически предоставляется всем пользователям Exchange Online. Дополнительные сведения см. в обзоре [электронной почты с кратким описанием](/Briefing/be-overview).

## <a name="microsoft-forms"></a>Microsoft Forms

Microsoft Forms позволяет быстро и легко создавать пользовательские тесты, опросы, анкеты, регистрации и т. д. После создания викторины или какой-либо формы можно предложить другим пользователям ответить на вопросы. При этом они могут использовать любой браузер, в том числе на мобильном устройстве. Данные форм, такие как результаты теста, можно легко экспортировать в Excel для дополнительного анализа или проверки. Дополнительные сведения см. [в Microsoft Forms?](https://support.office.com/forms).

Мобильное приложение Kaizala позволяет пользователям отправлять мгновенные сообщения и вложения, такие как рисунки, документы, видео, звуковые файлы и т. д., а также использовать карточки действий для опросов, опросов, заданий, объявлений, учебных заданий и т. д. ## <a name="microsoft-lists"></a>Microsoft Lists

Microsoft Списки — это Microsoft 365, которое помогает отслеживать информацию и упорядочивать работу. Готовые шаблоны позволяют быстро запускать списки в Интернете, в новом мобильном приложении или Microsoft Teams. Так как он является частью Microsoft 365, вы можете полагаться на безопасность и соответствие требованиям, готовым к использованию на предприятии.

## <a name="onenote-class-notebook"></a>Записная книжка OneNote для занятий

Записная книжка для занятий является частью OneNote доступна для Office 365 для образования и Office 365 E5 образования. Каждая записная книжка включает библиотеку содержимого, в которой преподаватели могут делиться содержимым курса, пространство для совместной работы, где преподаватели и учащиеся могут работать вместе, а также частную записную книжку для каждого учащегося. После создания записной книжки для занятий преподаватели и учащиеся могут использовать OneNote для доступа к ней с любого устройства.

## <a name="microsoft-planner"></a>Планировщик Microsoft

"Планировщик" (Майкрософт) — это инструмент, предоставляющий пользователям наглядный способ организации командной работы. Команды могут создавать планы, формировать и назначать задачи, делиться файлами, обсуждать ход выполнения работы, устанавливать сроки выполнения и обновлять статус. Кроме того, "Планировщик" позволяет связывать документы с конкретными задачами, совместно редактировать их и открывать беседы по задачам. Дополнительные сведения см. [в Планировщик (Майкрософт)](https://go.microsoft.com/fwlink/?LinkID=718016&amp;clcid=0x4809).

## <a name="microsoft-power-apps"></a>Microsoft Power Apps

Microsoft Power Apps — это корпоративная служба, которая помогает быстро создавать, интегрировать и совместно использовать приложения, которые работают на любом устройстве. Power Apps позволяет использовать встроенные подключения или подключения, созданные вашей компанией, для подключения приложения к облачным и локальным службам, включая Microsoft 365 и Office 365, Dynamics CRM, OneDrive, SharePoint, SQL Server, базы данных Oracle и т. д. Вы можете легко поделиться своим Power Apps с коллегами, введя адрес электронной почты. Дополнительные сведения см. на веб-сайте Power Apps [Майкрософт](https://powerapps.microsoft.com/).

Microsoft Power Automate позволяет автоматизировать рабочие процессы в приложениях. Flow позволяет подключать оповещения об электронной почте и мгновенных сообщениях, синхронизировать файлы между приложениями, копировать файлы из одной службы в другую, собирать данные из одного приложения и хранить их в другом и т. д. Для начала работы доступны шаблоны. Дополнительные сведения о Flow и их использовании см. на Power Automate [веб-сайте](https://preview.flow.microsoft.com/).

## <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream — это корпоративная видеослужб, которую сотрудники вашей организации могут использовать для безопасной отправки, просмотра и совместного использования видео. Вы можете поделиться записями занятий, собраний, презентаций, учебных семинаров и другими видеофайлами, которые будут полезны для совместной работы вашей команды. Microsoft Stream также упрощает общий доступ к комментариям к видео и пометить коды времени в комментариях и описаниях для ссылки на определенные моменты в видео.
  
## <a name="microsoft-sway"></a>Microsoft Sway

Microsoft Sway — это профессиональное приложение для создания цифровых историй для бизнеса, которое помогает вам и вашим коллегам выразить идеи с помощью интерактивного веб-холста. Встроенная подсистема проектирования Microsoft Sway помогает создавать профессиональные, визуально привлекательные отчеты, презентации и т. д. без необходимости расширенного форматирования или дополнительного обучения. Вы также можете изменить результаты, чтобы получить уникальный внешний вид и удобство. Microsoft Sway делает ваше создание отличным в любом браузере на любом экране, и его можно предоставить коллегам и клиентам, отправив ссылку. Microsoft Sway помогает находить и извлекать содержимое всех типов, не выходя из приложения, поэтому вы можете перетаскивать изображения, текст, видео и диаграммы прямо на холст. Дополнительные сведения см[. в Sway](https://sway.com).

## <a name="windows-update-for-business-deployment-service"></a>Служба развертывания Windows Update для бизнеса

Служба клиентский компонент Центра обновления Windows для бизнеса обеспечивает контроль над утверждением, планированием и защитой содержимого, предоставляемого клиентский компонент Центра обновления Windows. Она предназначена для работы с существующими политиками клиентский компонент Центра обновления Windows для бизнеса, чтобы обеспечить расширенный контроль над отдельными обновлениями. Служба развертывания предоставляет следующие возможности:

* Планирование развертывания обновлений компонентов на определенную дату
* Поэтапное развертывание в течение нескольких дней или недель с использованием форматированных выражений
* Обход предварительно настроенных политик клиентский компонент Центра обновления Windows для бизнеса, чтобы немедленно развернуть обновление системы безопасности в организации
* Обеспечение покрытия оборудования и программного обеспечения в организации с помощью развертываний, адаптированных для уникальной совокупности устройств
* Автоматическое определение и приостановка развертываний на устройствах, на которые может повлиять удержание  [asafeguard](/windows/deployment/update/safeguard-holds)

Эти возможности доступны через API microsoft Graph и интеграции с Microsoft Endpoint Manager. Дополнительные сведения см[. в клиентский компонент Центра обновления Windows службы развертывания для бизнеса](https://aka.ms/wufb/deployment-service).
