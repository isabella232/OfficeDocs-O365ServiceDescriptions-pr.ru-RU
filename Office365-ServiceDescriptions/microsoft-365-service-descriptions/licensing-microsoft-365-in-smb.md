# <a name="licensing-microsoft-365-for-small-and-medium-sized-businesses"></a>Лицензии Microsoft 365 для малого и среднего бизнеса

Microsoft 365 бизнес премиум является лучшим в своем классе предложением Для клиентов малого и среднего бизнеса (SMB). Microsoft 365 бизнес премиум это комплексное решение, которое  объединяет производительность Microsoft 365 с расширенным **управлением** безопасностью и **устройствами.**

В этой статье сравнивают параметры лицензирования для малых и средних предприятий с особым упором на Microsoft 365 бизнес премиум.

<sup>1</sup> Позволяет организации нажимать определенные параметры конфигурации (параметры безопасности, параметры приложения, роуминг Active Directory и так далее) для Office машин, войдя в корпоративную сеть.

<sup>2</sup> С Office стандартный 2016 и Office профессиональный плюс 2016 ИТ-система полностью контролирует обновления с помощью модели MSI. С Приложения Microsoft 365 для предприятий и Приложения Microsoft 365 для бизнеса, установка и обновления управляются с помощью click-to-Run.

<sup>3</sup> Наборы обеспечивают различные уровни интеграции с серверами производительности бизнеса.

<sup>4</sup> Архивная электронная почта Outlook не требуется .pst-файлов и установите политики хранения, которые автоматически архивировать &mdash; &mdash; или удалять электронную почту. ИТ-администраторы могут централизованно управлять In-Place Archive.

<sup>5</sup> Требуется дополнительная онлайн-служба(ы).

<sup>6</sup> Разрешает локализованную установку с помощью щелчка мыши в запуск лицензированных Microsoft 365 приложений, включенных в службу. Не включает права на установку Office локальной версии 2016. Office поддерживает компьютеры и Windows планшеты с Windows 7 или более, а macs с MAC OS X 10.6 или выше. Office для iPad можно установить на iPads с версией 7 или выше.

<sup>7</sup> Эта функция премиум-класса включена в Microsoft 365 бизнес премиум, но не в других Приложения Microsoft 365 для бизнеса предложениях.
