# <a name="visio-for-the-web"></a>Visio в Интернете

Visio для интернета расширяет возможности Microsoft Visio в браузере, где можно создавать и редактировать диаграммы, сохраненные в облаке. Клиенты Microsoft 365 могут просматривать, печатать и делиться диаграммами и вставлять комментарии в перейти. Если у вас есть подписка Visio, можно использовать дополнительные функции, например описанные в следующих разделах.
  
## <a name="apply-rich-formatting-to-text-and-shapes"></a>Применение форматирования к тексту и фигурам

Функции редактирования, такие как больше цветов, градиентов, эффектов и стилей, доступны в Visio для интернета, а также в настольном приложении. Кроме того, можно добавить текстовое поле или выбрать из галереи фигур и применить стили, которые определяют эффекты заполнения, контура и тени.
  
## <a name="arrange-objects"></a>Arrange objects

Вы можете перемещать, размерировать, вращать, переворачивать или заказывать фигуры и текстовые поля, а также разгруппировать фигуры для работы с ними в Visio для веб-сайта.
  
## <a name="build-mashup-solutions"></a>Создание гибридных решений

API Mashup Visio для веб-javaScript позволяет разработчикам получать доступ к веб-схеме Visio, ее страницам и ее фигурам и управлять ими. Дополнительные сведения см. [в веб-странице Visio: часто задающие вопросы.](https://support.office.com/article/e6647040-2fca-42ec-9fa5-d16a4e39e0ee)
  
## <a name="create-diagrams"></a>Создание схем

Вы можете легко создавать схемы в Visio для интернета и сохранять их в SharePoint или OneDrive для бизнеса. Но вы можете создавать схемы в классическом приложении Visio, отправлять их в SharePoint или OneDrive для бизнеса, а затем делиться ими.
  
## <a name="cut-copy-and-paste"></a>Команды "Вырезать", "Копировать" и "Вставить"

Функции cut, copy и paste доступны в Visio для интернета. Эти функции можно использовать для перемещения или удаления фигур и текста в схеме.
  
## <a name="edit-diagrams"></a>Редактирование схем

Visio для веб-сайта содержит богатое редактирование, которое можно использовать для внесения изменений в схему. Вы также можете изменить дизайн схемы с помощью различных цветов и макетов темы.
  
## <a name="embed-diagram-in-a-sharepoint-web-or-blog-page"></a>Внедрение схемы в страницу SharePoint, веб-сайта или блога

Вы можете использовать Visio для веб-страницы, чтобы вставить схему Visio (созданную в настольном приложении Visio) на странице SharePoint, веб-страницы или блога. Дополнительные сведения см. [в веб-странице Visio: часто задающие вопросы.](https://support.office.com/article/e6647040-2fca-42ec-9fa5-d16a4e39e0ee)
  
## <a name="find"></a>Найти

В Visio для веб-страницы вы можете искать каждое появление слова, номера или фразы на странице, выбрав **Кнопку More** (кнопка эллипсов), а затем выберите **Find** (или нажмите кнопку Ctrl+F). Результаты появятся рядом со схемой. Visio выполняет поиск в именах, тексте и данных фигур, а также в определенных пользователем ячейках.
  
## <a name="font-formatting"></a>Форматирование шрифта

Вы можете изменить шрифт, размер шрифта и другие атрибуты внешнего вида шрифта (такие как смелые, италики, суперсписки или подстройки) в Visio для веб-сайта.
  
## <a name="hyperlinks"></a>Гиперссылки

В Visio для интернета вы можете напрямую получить доступ к гиперссылкам, выбрав фигуру.
  
## <a name="navigation"></a>Навигация

В Visio для веб-страницы можно перейти на другие страницы, выбрав вкладки в левом нижнем.
  
## <a name="office-add-ins"></a>Надстройки Office

Надстройки Office еще не доступны в Visio для интернета.
  
## <a name="offline-viewing"></a>Просмотр в автономном режиме

Недоступна в Visio для интернета. Office для интернета требует подключения к Интернету и браузера. Для просмотра и редактирования схемы, отключенной от Интернета или локального сервера Office Web Apps Server, необходимо установленное на компьютере настольное приложение Visio. Если установлено настольное приложение Visio, вы можете использовать полную функциональность Visio для просмотра и редактирования схемы в автономном режиме. При сохранении внесенные вами изменения добавляются в схему в Интернете. Вы также можете скачать и сохранить автономную копию схемы. Дополнительные информацию о Office Web Apps Server см. в странице [Office Web Apps Server.](/webappsserver/how-office-web-apps-work-on-premises-with-sharepoint-2013)
  
## <a name="print-to-pdf"></a>Print to PDF

Вы можете распечатать в ФОРМАТЕ PDF или принтеры с помощью Visio для веб-страницы.
  
## <a name="proofing-tools"></a>Средства проверки правописания

Средства проверки, такие как проверка орфографии, перевода и настраиваемые словари, недоступны в Visio для Веб-сайта. Однако доступны проверка доступности и подробные текстовые описания.
  
## <a name="real-time-co-authoring"></a>Real-time co-authoring

Еще не доступен в Visio для веб-сайта. С помощью Visio в Интернете можно просматривать, добавлять или удалять комментарии.
  
## <a name="save-as-or-download-a-copy"></a>Save As or Download a copy

Visio для веб-сайта позволяет использовать Save As или скачивать копию схемы, но для редактирования локальной копии необходимо иметь настольное приложение Visio.
  
## <a name="share-a-diagram"></a>Общий доступ к схеме

Если схема сохранена в библиотеке документов SharePoint или в OneDrive для бизнеса, она уже находится в сети, поэтому к ней можно предоставить доступ, отправив по электронной почте не само вложение, а лишь ссылку. Пользователи с соответствующими разрешениями смогут просматривать ее в веб-браузере или на мобильном устройстве. Просто выберите **Share** в правом верхнем справа. Рекомендуется использовать ссылку, чтобы избежать создания дублирующих схем, связанных с данными.
  
## <a name="smartart"></a>SmartArt

С помощью Visio для веб-страницы вы не можете добавить SmartArt в схему. Однако если smartArt был добавлен с помощью настольного приложения Visio, оно будет видно в Visio для веб-сайта.
  
## <a name="tell-me"></a>Помощник

Функция Tell Me недоступна в Visio для интернета. В классическом приложении Visio с помощью функции помощника можно быстро находить справку по нужным задачам.
  
## <a name="undo-and-redo"></a>Отмена и повтор

Функции Undo и Redo доступны в Visio для интернета.
  
## <a name="view-and-add-comments"></a>View and add comments

 В Visio для интернета вы можете сотрудничать с другими, просматривая, добавляя и реагируя на комментарии в схеме. 
  
## <a name="view-shape-data"></a>Просмотр данных фигур

Вы можете использовать Visio для веб-страницы, чтобы увидеть метаданные для схемы: просто выберите фигуру, а затем выберите **Shape Info**.
  
## <a name="view-visio-files-in-the-browser"></a>Просмотр файлов Visio в браузере

Так как схема Visio сохраняется в Интернете, вы можете отправить ссылку на нее. После этого другие пользователи смогут просматривать ее в любом браузере. Для просмотра схемы в браузере не требуется устанавливать настольное приложение Visio или лицензию Visio.
  
## <a name="zoom"></a>Масштабирование

Visio для веб-сайта включает ползунок, который позволяет увеличить и выйти. Кроме того, для масштабирования области диаграммы можно использовать функцию Pan &amp; Zoom.
